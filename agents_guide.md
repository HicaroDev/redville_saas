# AGENT MASTER PLAN – REDVILLE SAAS

## Objective
Build a modular SaaS for construction management.

## Development Strategy
- MVP first
- Modular architecture
- Iterative evolution

## Phases

### Phase 1 – Foundation
- Auth
- Project creation
- Cost centers

### Phase 2 – Financial Core
- Cost input
- Cash flow
- Basic dashboard

### Phase 3 – Intelligence
- KPIs
- Progress tracking

### Phase 4 – Expansion
- Inventory
- Purchases
- RDO

## Rules

- Do NOT over-engineer
- Always validate before moving
- Keep UI simple
- Prioritize usability

## Architecture Principles

- Separation of concerns
- Component reuse
- API-first mindset

## Data Model (High Level)

- Users
- Projects
- Cost Centers
- Transactions
- Progress Logs

## Key Risk

- Complexity creep

## Mitigation

- Always build smallest version first
